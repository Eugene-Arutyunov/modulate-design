# Документация: Исследование айдентики компаний

## Описание проекта

Проект предназначен для исследования визуальной айдентики (брендинга) ведущих AI-стартапов и технологических компаний мира.

## Структура проекта

- **Страница исследования**: `src/identity-research.html` (все тексты на странице должны быть на английском языке)
- **Графика**: `src/assets/identity research/`
- **Документация**: `IDENTITY-RESEARCH.md` (этот файл)

## Язык интерфейса

**Важно**: Вся страница `identity-research.html` должна быть на английском языке. Все заголовки, тексты и интерфейсные элементы должны быть переведены на английский.

## Задания

### Задание №1: Список компаний

Сформирован список из:
- **30 топовых AI-стартапов**
- **20 топовых технологических компаний**

Список записан непосредственно на странице `identity-research.html` и может редактироваться вручную. Изменения в списке являются сигналом для обновления другого контента (атрибутов компаний, графики и т.д.).

## Процесс работы

1. **Список компаний** - основной источник данных, редактируется вручную
2. **Поиск атрибутов** - по списку компаний собираются различные атрибуты (логотипы, цвета, типографика и т.д.)
3. **Обновление контента** - при изменении списка компаний автоматически обновляется связанный контент

## Формат данных

Компании представлены в виде структурированного HTML-списка на странице исследования.
