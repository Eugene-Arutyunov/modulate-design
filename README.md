# Modulate Design

Static site built with Eleventy for design system documentation and demos.

## Requirements

- Node.js 18+
- npm

## Quick Start

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Eleventy

[Eleventy](https://www.11ty.dev/) is a static site generator. It takes files from `src/`, processes templates, and generates a ready-to-deploy site in `_site/`.

## Structure

- `src/` — source files
- `_site/` — built site
- `eleventy.config.js` — Eleventy configuration
