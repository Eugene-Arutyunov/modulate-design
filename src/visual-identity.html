Industry Legibility vs Authenticity

The task of developing our visual language is to balance clear identification within the competitive landscape with the creation of a memorable and authentic Modulate expression.

Industry Legibility means Modulate:

  • looks like a tech company,
  • looks like an AI company,
  • looks relevant for 2026.

Achieved through aligning with core visual norms established by key industry players.

Authenticity means Modulate:

  • looks like Modulate,
  • feels modulateish,
  • conveys Modulate vibes.

Achieved by deliberately searching outside our category to surface visual cues that clearly differentiate Modulate. This exploration is guided by a light awareness of existing references and anchored in our core brand values.

Good examples from industry leaders.

Characteristics of industry legibility:

  • the primary color is black or dark
  • the graphics are very simple
  • most wordmarks use low-contrast closed grotesques — a core 2020s trend. A rare and nice exception is ElevenLabs, which opts for a higher-contrast grotesque.

There has to be a square element

The logo needs a square element alongside the wordmark; without it, we have to make up a new icon for every platform, app badge, profile picture, etc., which hurts consistency. A built-in square mark gives us a clear, repeatable icon wherever a square format is needed.

The logo doesn’t solve the icon problem — it forces a new symbol to be invented each time, which breaks recognition and consistency.

The logo is just text, so a separate symbol ends up serving as the icon — like OpenAI, where Blossom isn’t part of the logo but inevitably becomes the main identifier.

The logo includes a square symbol, which naturally becomes the default icon whenever a square format is required.
